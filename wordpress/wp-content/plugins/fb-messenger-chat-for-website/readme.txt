Integrate Facebook Messenger Chat on your website easily without touching any code.

== Description ==
Facebook is one of the best and biggest social media platform, millions of people are on facebook now. It is the greatest way to connect with your friends and customers. It\'s easy and free!

Use our Facebook Messenger Chat for Website Plugin to integrate FB Messenger Chat on your website without touching a single line of code.

= Features =
* Easy and Simple, you just need to add your facebook page ID.
* You can change the position of the floating chat badge from right to left or left to right.
* You can change the theme color of the chat box and chat badge to match it with your theme.

= Common Issues =
* If you're not able to see the badge on your website even after adding your page ID, please add your website in your Facebook Page's Whitelist. Check our FAQ Section for more details.

== Installation ==

Installation
= Way 1: =
1. Upload plugin to the /wp-content/plugins/ directory
2. Extract the zip file.
3. Activate the plugin from wordpress plugins page. And you have done.

= Way 2: =
1. Go to plugins page in admin.
2. Click on Add New button.
3. Search for “Facebook Messenger Chat for Website”.
4. Click install on Facebook Messenger Chat for Website.
5. Activate the plugin

== Frequently Asked Questions ==

= Why Facebook Messenger Badge is not showing on my website? =

Please add your website URL in your facebook page's whitelist.
Go to your facebook page settings and then click on Messenger Platform, find whitelist your domain and add your website's URL.
